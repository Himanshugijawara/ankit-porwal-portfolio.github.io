## 🔧 How the Dynamic Sync Works

Every time someone visits the portfolio:

1. 🤖 The page calls the **Claude AI API**
2. 🔍 Claude searches your **Behance profile** for all projects
3. 🖼️ Projects are fetched with names, links & cover images
4. ✨ They render **automatically** on the portfolio — no code changes needed!

> **Add a new project to Behance → It appears on your portfolio automatically!**
